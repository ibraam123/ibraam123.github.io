My Projects

Math Game

This project is a math quiz game where users can select the number of questions and difficulty level (easy, medium, hard, or mixed). It generates random arithmetic problems (addition, subtraction, multiplication, or division), checks the answers, tracks correct and incorrect responses, and provides a pass /fail result at the end.

Stone-Paper-Scissors

This project is a Rock, Paper, Scissors game where the player competes against the computer. The game allows the user to select the number of rounds (between 1 and 10), and in each round, the player and computer choose either rock, paper, or scissors. The results of each round are displayed, and after all rounds, the game announces the winner based on the number of rounds won by each. The player can also choose to play again after the game ends.

Client Management System

This project is a client management system in C++ that allows users to add, update, delete, and search for client records stored in a text file. It includes functions for managing client data, such as validating account numbers, displaying client information, and marking clients for deletion rather than removing them immediately. The system is menu-driven, offering a simple interface for interacting with the client records.
